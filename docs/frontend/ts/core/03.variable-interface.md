# 类型声明及推断

## 类型声明

> 对变量或者函数形参进行类型声明

### 变量

```ts
let a: string;
let b: number;
let c: boolean;

// a = 9
// a = false
a = "Halo World";
b = 99;
c = true;
```

### 函数

```ts
function count(x: number, y: number): number {
  return x + y;
}

let res = count(1, 2);
console.log(res);
```

### 字面量

```ts
// 在 : 后也可以写字面量类型, 不过实际开发中很少使用
let x: 'Halo'
let y: 100

x = 'Halo'
b = 100
```

## 类型推断

> `TS`会根据代码进行类型推导

```ts
let a = 99

// a = 'Halo'
a = 1
```

⚠️ 注意: 类型推断不是万能的, 面对复杂类型时推断容易出问题, 所以尽量明确的编写类型声明
