# 前言

当学习了其他语言后再用`JavaScript`确实感觉有很多痛点

比如写错单词编辑器没有提示警告, 自定义的对象方法调用自身属性时没有语法提示等

习惯了`JavaScript`再用`TypeScript`刚开始有些不适应, 用多了就会感觉`TypeScript`真香

跟着本文档学习, 让我们一起拥抱`TypeScript`吧!
