# 前端学习路线

前端开发是指创建网站或Web应用程序用户界面(UI)的部分，主要涉及以下技术：

## 核心技术

- **HTML** - 网页内容结构
- **CSS** - 样式和布局设计
- **JavaScript** - 交互和动态功能

## 现代前端技术栈

- **框架/库**：React, Vue, Angular, Svelte
- **CSS预处理器**：Sass, Less
- **构建工具**：Webpack, Vite, Rollup
- **包管理器**：npm, yarn, pnpm
- **TypeScript** - JavaScript的超集，添加类型系统

## 进阶领域

- **响应式设计** - 适配不同设备
- **Web性能优化**
- **PWA(渐进式Web应用)**
- **WebAssembly**
- **前端测试** (Jest, Cypress等)
